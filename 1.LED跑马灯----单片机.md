## （2）74HC573锁存器

GND接地->0;

## （3）任务实现

建立Keil5 

1.先建立新的文档，最好提前建立好。

2.建立新的文本，保存到该文档中，并以.c为命名。

代码实现：

| 闪烁三次，后一个个产生灯，再一个个熄灭灯。 |
| ------------------------------------------ |



```
#include "reg52.h"

sbit HC138_A = P2^5;
sbit HC138_B = P2^6;
sbit HC138_C = P2^7;

void Delay(unsigned int t)
{
	while(t--);
	while(t--);
}

void LEDRunning()
{
	unsigned char i;
	HC138_C = 1;
	HC138_B = 0;
	HC138_A = 0;
	

for(i = 0; i < 3; i++)
{
P0 = 0x00;
Delay(60000);
Delay(60000);
P0 = 0xff;		//ff为关闭
Delay(60000);
Delay(60000);
}

for(i = 1; i <= 8; i++)
{
	P0 = 0xff << i;    //取依次打开一个灯 向左。
	Delay(60000);
	Delay(60000);
}

for(i = 1; i <= 8; i++)
{
	P0 = ~(0xff << i); //取反
	Delay(60000);
	Delay(60000);
}

}

void main()
{
	while(1)
	{
		LEDRunning();
	}
}
```
